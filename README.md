# Clinical Trial Data Analysis

SQL-based analysis of clinical trial data using Apache Spark SQL, exploring treatment outcomes, patient demographics, and trial performance metrics.

## Overview

This project applies Spark SQL to a clinical trial dataset to uncover patterns in patient outcomes, treatment efficacy, and trial progression — demonstrating distributed query processing for healthcare data at scale.

## Contents

| File | Description |
|------|-------------|
| `Analyzing Clinical Trial Data Using Spark SQL.sql` | Core SQL queries for the analysis |
| `data/` | Input datasets used in the analysis |

## Analysis Highlights

- Patient demographic breakdowns
- Treatment group comparisons
- Outcome frequency and distribution analysis
- Trial progression and completion rates
- Aggregate metrics across trial phases

## Getting Started

### Prerequisites
- Apache Spark with Spark SQL support
- Databricks or a local Spark environment

### Running the Analysis
1. Clone the repository
2. Load the datasets from the `data/` folder into your Spark environment
3. Run the SQL script in a Spark SQL context or Databricks notebook
